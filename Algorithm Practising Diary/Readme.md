双向指针算法练习心得
 
双向指针最核心的价值在于简化遍历逻辑、降低时间复杂度。比如做“三数求和”“四数之和”时，相比暴力嵌套循环，固定基准后用左右指针从两端收缩，能将O(n³)降至O(n²)，高效且易实现。
 
其适用场景有规律：一是有序数组（如“两数求和”“最接近的三数之和”），有序性可通过数值比较精准调整指针方向；二是区间最优解问题（如“盛水最多的容器”），通过移动短板指针缩小范围，避免全量枚举。
 
练习中需注意细节：比如“三数求和”要同时对基准元素和双指针做去重处理；“搜索旋转排序数组”需先判断mid两侧的有序性再移指针。另外双指针并非固定“左右对撞”，像“接雨水”的双向指针法，通过记录左右最大高度灵活计算，体现了其变形思路的实用性。
