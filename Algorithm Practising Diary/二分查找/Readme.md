二分查找算法练习心得
 
二分查找的核心是通过不断缩小“可能存在目标的区间”高效定位结果，适用于“有序”或“存在明确边界的搜索场景”，能将线性查找的O(n)复杂度降至O(logn)，是处理大规模数据查找的高效工具。
 
从练习的题目来看，其适用场景可归为两类：
 
- 直接查找类：如“在排序数组中查找元素的第一个和最后一个位置”，利用二分找边界（左边界时遇相等不停止，继续左移；右边界则相反）；“H指数II”通过二分判断 citations[mid] 与“剩余元素数”的关系定位结果。
- 二分答案类：像“爱吃香蕉的珂珂”“完成旅途的最少时间”，先确定答案的上下界（如珂珂的最小/最大吃香蕉速度），再通过“是否满足条件”（能否在规定时间吃完）收缩区间，这类题的关键是“把‘求结果’转化为‘判断某个值是否可行’”。
 
练习中需注意细节：一是明确“二分条件”，比如“统计公平数对的数目”中，需通过二分找到符合“nums[j]范围”的边界，条件写错会导致区间收缩错误；二是处理“边界值”，比如“正整数和负整数的最大计数”中，二分找到0的位置后，需准确计算左右元素数量，避免漏算或多算。
